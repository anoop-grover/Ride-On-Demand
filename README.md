
# 🚗 Ride-On-Demand

Welcome to **Ride-On-Demand**, your ultimate full-stack car rental platform!  
Seamlessly book, manage, and explore vehicles with 3 unique modules: **User, Admin, and Vendor**.  
Split into **Client** and **Backend**, powered by modern technologies for smooth operations.  



## 🌟 Installation

Clone Rent a Ride

```bash
https://github.com/anoop-grover/Ride-On-Demand.git
```

Install node modules

```bash
  cd backend
  npm install
  npm run dev
```
```bash
  cd client
  npm install
  npm run dev
```

    
## 🛠 Tech Stack

**Frontend:** React, Redux Toolkit, Tailwind CSS, Material UI, React Toast, React Hook Form, Zod, Google OAuth, Razorpay

**Backend:** Node.js, Express.js, MongoDB, Cloudinary, Nodemailer, Multer, JWT auth, Role-based access

**Deployment:** AWS EC2, Nginx (reverse proxy), Cloudflare (DNS), PM2 (uptime)


## 🚀 Project Description
Rent a Ride isn’t just a car rental platform — it’s a full-scale vehicle ecosystem!

Whether you’re a User, Admin, or Vendor, everything you need is just a click away.

##

**Key Features & Modules:**

**👤 User Module:**

* Browse & book vehicles online ✨
* Manage profile & account settings 📝
* Track past & upcoming bookings 🕒
* Receive email notifications on bookings 📧

**🛡 Admin Module:**

* Manage bookings, vehicles & users efficiently ⚡

* Approve or remove vendors ✅

* Oversee the platform for smooth operations 🌐

**🚚 Vendor Module:**

* Dedicated sign-up/sign-in flow 🔑

* Add vehicles for admin approval 🏎️

* Receive real-time booking notifications 🔔
  
##

##

**💡 Features & Implementations:**

* JWT Authentication – Secure login flows for all roles 🔒

* Role-Based Access – Protected routes for Admin, User & Vendor ⚙️

* Dynamic Location Selector – Pickup & drop-off options auto-update 🌍

* Search, Filter, Sort – Find your ride in seconds 🔍

* UI Built from Scratch – Sleek, responsive, interactive ✨

* Google OAuth – Quick & secure sign-ups 🟢

* Email Notifications – Automated booking confirmations 📧

* Cloudinary Integration – Optimized media storage ☁️

* MongoDB Optimizations – Referenced models & aggregation pipelines 📊

* File Uploads with Multer – Vehicle images/videos upload 📸

* Version Control with Git – Collaboration & backup 🗂️
